# クイック改善プロンプト

すぐに使える短縮版プロンプトです。Claude Codeに貼り付けて実行してください。

---

## 📋 コピー&ペースト用プロンプト

```
このリポジトリ内の全てのJupyter Notebookを、日本語の教科書フレームワークに基づいて改善してください。

【必須要素】
各ノートブックに以下を追加：
- 学習目標（チェックボックス形式）
- 前提知識、推定時間、難易度
- 詳細な日本語説明（10,000文字以上）
- コード内の詳細なコメント（200行以上）
- 実世界の応用例とアナロジー
- よくあるエラーと解決法（2-3個）
- 自己評価クイズ（2-5問）
- 次のステップ

【構造】
Why → What → How → Practice → Summary の順で構成

【禁止事項】
❌ 進捗管理機能（UI要素）は追加しない
❌ プレースホルダーを残さない
❌ 元のコードの動作を変更しない

【必須事項】
✅ コード内に200行以上の詳細なコメント
✅ 概念の前に詳しい日本語説明
✅ ユーザーが概念学習に集中できるよう、実装詳細はコメントで説明

【実行手順】
1. まず全ノートブックを確認して分析・計画を提案
2. パイロット実装（1つのノートブック）
3. フィードバック反映後、全体に適用
4. プレースホルダー削除
5. リポジトリ整理（scripts/フォルダ作成）

段階的に進め、各ステップで確認を取ってください。
```

---

## 使い方

1. 上記のプロンプトをコピー
2. 改善したいノートブックリポジトリで Claude Code を起動
3. プロンプトを貼り付けて実行
4. 各フェーズで確認しながら進める

**推定時間**: 2-4時間（ノートブック数による）
