# サンプルスクリプト集

このディレクトリには、機械学習の基本的な概念を学ぶためのシンプルなサンプルスクリプトが含まれています。

## 📋 ファイル一覧

### 機械学習アルゴリズム

#### `linearRegression.py`
線形回帰の基本的な実装例。

**内容:**
- 線形回帰モデルの作成
- データのフィッティング
- 予測の実行

---

#### `naivebayes.py`
ナイーブベイズ分類器の実装例。

**内容:**
- テキスト分類
- ナイーブベイズアルゴリズムの基本
- 確率計算

---

#### `perceptlon.py`
パーセプトロンアルゴリズムの実装例。

**内容:**
- 単純パーセプトロン
- 二値分類
- 重みの更新

---

#### `tensorflow_intro.py`
TensorFlowの入門スクリプト。

**内容:**
- TensorFlowの基本的な使い方
- テンソル操作
- 簡単なニューラルネットワーク

---

### データ処理

#### `morphological.py`
形態素解析のサンプル。

**内容:**
- テキストの形態素解析
- 自然言語処理の基本
- トークン化

---

#### `readhtmlpandas.py`
PandasでHTMLを読み込むサンプル。

**内容:**
- `pandas.read_html()` の使用例
- WebスクレイピングHTMLテーブルのデータフレーム化

---

### ユーティリティ

#### `loggingexample.py`
Pythonのロギング機能のサンプル。

**内容:**
- ロギングの基本設定
- ログレベルの使い分け
- ファイルへのログ出力

---

## 🚀 使用方法

各スクリプトは独立して実行できます：

```bash
python linearRegression.py
python naivebayes.py
# など
```
